## 🚴 Customer Bike Buying Insights (Excel Dashboard)
## 📌 Project Overview
The Customer Bike Buying Insights Dashboard is an Excel project that helps analyze which type of customers are more likely to buy bikes.
This dashboard shows useful information about customer income, education, marital status, gender, and region using charts and filters.
It helps understand customer behavior and bike sales trends.
## 🎯 Objective
The goal of this project is to:

1.Understand customer buying behavior

2.Find out which customers purchase more bikes

3.Show insights using charts and visuals

4.Create an interactive Excel dashboard

## 📊 Dashboard Features
1.💰 Total Income – Shows total income of customers

2.🚲 Bikes Purchased – Total number of bikes purchased

3.🎓 Education Analysis – Comparison of educated customers

4.👨‍👩‍👧 Marital Status Analysis – Married vs single customers

5.🌍 Region Filter – Europe, North America, Pacific

6.💼 Occupation Filter – Different customer job types

7.📈 Charts & Graphs – Bar chart, line chart, pie chart

8.🎛 Slicers – Interactive filters for better analysis

## 📈 Key Insights
1.Customers with higher income buy more bikes.

2.Married customers buy more bikes than single customers.

3.Education level affects bike buying behavior.

4.Gender and occupation also influence purchases.
